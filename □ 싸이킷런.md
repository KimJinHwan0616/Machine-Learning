## 데이터 생성
+ ### 분류
    ```
    from sklearn.datasets import make_classification
    
    x, y = make_classification(
        n_samples = 표본 수
        , n_features = 독립변수 개수
        , n_informative = 독립변수 개수       # 종속변수와 상관관계 
        , n_redundant = 독립변수 개수         # 독립변수끼리 종속관계
        , n_clusters_per_class = 군집수
        , suffle = False
        )
    ```
+ ### 군집
  정규분포
    ```
    from sklearn.datasets import make_blobs
    
    x, y = make_blobs(
        n_samples = 표본 수
        , centers = 군집 수
        , cluster_std = 군집 표준편차
        )
    ```
