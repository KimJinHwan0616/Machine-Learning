## 범주형(질적) 데이터
+ ### 순서 데이터
    문자열 → 인덱스
    ```
    from sklearn.preprocessing import LabelEncoder
    
    def label_encoder(table):
      
        le = LabelEncoder()    # 라벨 인코더 생성
    
        features = ['속성_리스트']
        for feature in features:
            table[feature] = le.fit_transform( table[feature] )    # 문자열 → 인덱스
        
        return table
    
    테이블_변수 = label_encoder( 테이블_변수 )    
    ```
